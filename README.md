# AI-Assignment-Marker

A Python application that assists teachers in marking student assignments using AI.

## Features

- Graphical user interface to allow easy user interaction and usage
- Select folder where student assignment submission are located
- Read in Assignment task sheet
- Read in Assignment criteria marking sheet
- Read in each student submission in a word document format in turn
- Assess each student submission against the task sheet and the criteria marking sheet
- Provide both feedback and a grade across each criteria
- Provide overall grade feedback together with an overall grade
- Provide a student copy of the feedback and the grades including overall grade.
- Store the students grade for each criteria and the overall grade in spreadsheet

## Teachnologies Used

- Python
- Visual Studio Code
- Tkinter
- OpenAI API
- OpenPyXL
- python-docx
